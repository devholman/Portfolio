Expanding Your Horizons

Graduation Day!

On Friday I graduated from a 12 week immersive coding bootcamp. The exposure I received to HTML, CSS, Javascript, and Javascript librarys was invaluable. The first couple of weeks was spent hammering out efficient ways to use HTML and CSS. I was also introduced to github and began getting practice creating and managing my own repositories. The next few weeks were spent gaining proficiency with functional programming in Javascript, such as, mapping, recursing, reducing, and pipelining. My experience learning Javascript was definitely a challenge, but I found it to be very gratifying once I was able to master these new concepts. Weeks 6-8 were spent learning Backbone.js and React.js libraries to help manage my projects that were growing in scale. I got practice pulling data from different API's and manipulating that data on my own sites. Learning to use data however I wanted was one of the coolest parts about learning how to code. I spent the last few weeks working on my final project, Roomies, a chore manageemnt app. Instead of using data from an API, I wanted to practice using data that a user would input. Roomies

Roomies was created to help members of a shared living space manage their chores. The app allows all members of a household to contribute to a shared chores list. Each member can then claim the chores for which they agree to complete. This allows for an easy and interactive way to divide and conquer all of the shared responsibilities of the household. With the Roomies app, sharing a home has never been easier. Click Here to see it in action.

Project Details

Tech used: Javascript, React, Backbone, jQuery, and Firebase



Simple To Do List

Simple To Do List has an easy to use design layout to help the user focus on the tasks at hand. Still, it has all of the core features that a to do list should. You can add items to the list, scratch them off when you are done, or remove them entirely. There is an "All", "To Do", and a "Done" view to make it easier to track your progress.
